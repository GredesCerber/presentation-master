# 🎓 Presentation Master

**Система генерации HTML-презентаций с речевыми заметками**

---

## 📦 Что это?

Presentation Master — агент для VS Code Copilot, который создаёт готовые к использованию презентации:

- **Входные данные**: тема + имена авторов
- **Выходные данные**: 2 HTML-файла (презентация + речевые заметки)
- **Слайдов**: 14-22 (автоматически рассчитывается по числу авторов)
- **Стилей**: 7 дизайнов (автоматически подбираются по теме)
- **Зависимости**: нет (всё встроено в HTML)

---

## 🚀 Быстрый старт

### Установка

```powershell
# Клонируйте репозиторий
git clone https://github.com/GredesCerber/presentation-master

# Запустите установщик
cd presentation-master
.\install.ps1
```

### Использование

Откройте VS Code → Copilot Chat → выберите агент `presentation-master` или напишите:

```
Создай презентацию про машинное обучение, авторы: Иван и Мария
```

Результат:
- `Машинное_обучение.html` — интерактивная презентация (15 слайдов)
- `Речь_Машинное_обучение.html` — полный текст выступления

## 🎨 Стили дизайна

### По умолчанию: Academic (зелёный/бирюзовый)

Все презентации создаются в стиле **Academic** — тёмный фон с зелёными/бирюзовыми акцентами. Это проверенный стиль из папки `examples/`.

### Альтернативные стили (только по запросу)

Чтобы использовать другой стиль, явно укажите его в запросе:

| Стиль | Как указать | Описание |
|-------|-------------|----------|
| **Glassmorphism** | "в стиле Glassmorphism" | Стекло + анимированные сферы |
| **Minimal Dark** | "тёмный минималистичный" | Чистый тёмный фон |
| **Minimal Light** | "светлый/белый" | Белый фон, официальный |
| **Gradient Mesh** | "градиент/цветной" | Динамичные градиенты |
| **Neomorphism** | "неоморфизм" | Мягкие тени |
| **Brutalist** | "брутализм/дерзкий" | Чёрный/красный |
| **Corporate** | "корпоративный" | Светло-синий, деловой |

**Примеры:**
```
# По умолчанию (Academic стиль)
Создай презентацию про машинное обучение, авторы: Иван и Мария

# С указанием стиля
Создай презентацию про стартапы в стиле Glassmorphism, авторы: Анна
Создай презентацию про медицину в светлом стиле, автор: Пётр
```

## 🎯 Возможности

### Типы слайдов (автоматически подбираются)

| Тип | Описание | Когда используется |
|-----|----------|-------------------|
| **Title** | Заголовок + авторы | Первый и последний слайд |
| **Content** | Заголовок + буллеты | Основной контент |
| **Two-Column** | Сравнение/контраст | Плюсы/минусы, до/после |
| **Card Grid** | 4-6 карточек | Этапы, категории, примеры |
| **Section Divider** | Переходный слайд | Между разделами |
| **Example** | Кейс/пример | Практические иллюстрации |

## 🐛 Устранение проблем

**Агент не появляется в списке?**
```
Shift+Ctrl+P → "Reload Window"
```

**Файлы НЕ создаются, а выводятся в чат?**
- Это была проблема v1.1 — исправлено в v1.2
- Агент теперь явно использует `create` tool
- Переустановите: `.\install.ps1 -Force`

**Нужно удалить и переустановить?**
```powershell
.\install.ps1 -Force
```
